# Personal Portfolio - Shivtej Devkar

A premium, modern, and fully responsive personal portfolio website showcasing expertise in MERN stack and full stack web development.

![Portfolio Screenshot](file:///d:/freelance%20portfolio/index.html) *(Preview locally by opening index.html)*

## 🚀 Features

- **Premium Glassmorphic Design**: Clean and modern aesthetics with subtle gradients, HSL colors, and micro-animations.
- **Dynamic Interactive Elements**:
  - Custom page loading screen with progress indicator.
  - Scroll progress bar at the top of the viewport.
  - Interactive typing text effect in the Hero section.
  - Animated stats counters for projects, APIs, and daily active users.
- **Detailed Skill Visualization**: Beautifully categorized progress indicators for Frontend, Backend, Databases, and Tools.
- **Projects Showcase**: Interactive cards featuring detailed descriptions and tech tags for key projects (e.g., *VideoTube*, *Hotel Booking System*).
- **Contact Form**: Sleek contact section for inquiries and collaborations.

## 🛠️ Built With

- **Structure**: Semantic HTML5
- **Styling**: Modern Vanilla CSS3 (Custom properties, Flexbox/Grid layouts, Glassmorphism, Responsive design)
- **Interactions**: Vanilla JavaScript (Animations, typing effects, and active navigation highlighting)

## 📂 Project Structure

```text
├── index.html     # Main page structure and HTML markup
├── style.css      # Core styles, glassmorphism design tokens, and media queries
├── script.js     # Scroll animations, typing effect, and interactivity
└── README.md      # Project documentation
```

## 💻 Getting Started

To view the portfolio locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Shivtej-07/Portfolio-01.git
   ```
2. Navigate into the directory and open `index.html` in your favorite web browser.
